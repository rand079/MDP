UKCrash dataset

This dataset uses data made available at https://data.gov.uk/dataset/cb7ae6f0-4be6-4935-9277-47e5ce24a11f/road-safety-data. The class variable is accident severity, against 18 predictor variables. Data is from 2010-2016 and accidents are ordered sequentially. The dataset is an ARFF file compressed into RAR format and can be uncompressed using WinRAR or a similar tool.

If you use this dataset for experiments, please cite our work:

R. Anderson, Y.S. Koh and G. Dobbie, "Predicting concept drift in data streams using metadata clustering", IJCNN, 2018
